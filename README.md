# Aithneen Visual Identity

Bilingual interactive visual identity book for Aithneen.

## Files

- `index.html` - Cloudflare Pages entry point
- `AITHNEEN_FULL_VISUAL_IDENTITY_BOOK.html` - named source copy
- `amoshref-thulth.ttf` - local Arabic heading font
- `AITHNEEN_VISUAL_IDENTITY_LOCK_V4.md` - visual identity specification
- `OFL.txt` - font license

## Local Preview

```powershell
python -m http.server 8765
```

Then open `http://127.0.0.1:8765/`.
